📖 项目背景
在测绘、空间规划及科研等领域，业务人员需要频繁查阅《全国标准信息公共服务平台》上的各类国家标准文件。传统的手动检索、点击、下载流程不仅耗时耗力，且难以形成结构化的本地文档库。

本项目旨在通过 Python 自动化脚本，实现基于业务关键词的定向检索与批量下载，将繁琐的资料收集工作交由程序完成，大幅提升工作效率。

✨ 核心特性 (Features)
🔍 关键词定向检索：支持输入特定行业关键词（如“测绘”、“地理信息”），自动抓取相关标准列表。
🛡️ 智能绕过防护：内置网络请求伪装与身份认证逻辑，有效应对目标网站的常见防反爬策略。
⚡ 稳健的下载机制：支持大文件流式下载，并在网络波动时保证下载的完整性。
📂 自动化本地管理：根据抓取到的元数据（标准号、发布日期等），自动对下载的 PDF 文件进行重命名与分类存储。
🤖 AI 赋能开发：本项目核心架构与复杂逻辑重构均借助大语言模型（LLM）高效完成，验证了 AI 辅助编程在自动化工具开发中的极高效率。
🛠️ 技术栈
语言：Python 3.x
核心库：requests (网络请求), BeautifulSoup / lxml (DOM 解析), os / shutil (文件系统操作)
🚀 快速开始 (Quick Start)
1. 克隆项目
git clone [https://github.com/edjsh175/cn-standard-downloader]
cd cn-standard-downloader
2. 安装依赖
建议在虚拟环境中运行：

Bash
pip install -r requirements.txt
3. 运行配置与启动
在 config.py 或脚本顶部配置你的目标关键词和保存路径：

Python
# 示例配置
KEYWORDS = ["测绘", "空间规划"]
SAVE_DIR = "./downloads"
运行主程序：

Bash
python main.py
